# Praktikum 04: Master Template

**Status:** ✅ Selesai

Tutorial membuat master template dengan layout inheritance menggunakan @extends dan @section di Laravel.

## 📖 Dokumentasi Lengkap

Baca dokumentasi lengkap di: **[docs/README.md](docs/README.md)**

## 🎯 Tujuan Pembelajaran

- Membuat master template layout
- Menggunakan template inheritance dengan @extends
- Menggunakan @section dan @yield
- Membuat struktur template yang reusable

## 🚀 Quick Start

```bash
# Masuk ke folder project
cd praktikum_04_master_template/praktikum_laravel

# Install dependencies
composer install

# Setup environment
cp .env.example .env
php artisan key:generate

# Jalankan server
php artisan serve
```

Buka browser: **http://localhost:8000/mahasiswa**
