# Praktikum 02: Routing & Controller

**Status:** ✅ Selesai

Tutorial membuat route dan controller di Laravel untuk menampilkan data mahasiswa.

## 📖 Dokumentasi Lengkap

Baca dokumentasi lengkap di: **[docs/README.md](docs/README.md)**

## 🎯 Tujuan Pembelajaran

- Membuat route di Laravel
- Memahami cara kerja controller
- Mengirim data dari controller ke view
- Membuat controller mahasiswa dengan data dummy

## 🚀 Quick Start

```bash
# Masuk ke folder project
cd praktikum_02_routing_controller/praktikum_laravel

# Install dependencies
composer install

# Setup environment
cp .env.example .env
php artisan key:generate

# Jalankan server
php artisan serve
```

Buka browser: **http://localhost:8000/mahasiswa**
