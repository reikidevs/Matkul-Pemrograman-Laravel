# Product Overview

This is a Laravel tutorial repository for the PBKK (Pemrograman Berbasis Kerangka Kerja / Framework-Based Programming) course at Universitas Semarang, taught by Ahmad Rifa'i, S.Kom., M.Kom.

## Purpose

Educational repository containing step-by-step Laravel tutorials organized as weekly praktikum (practical exercises). Each praktikum builds incrementally on the previous one, teaching Laravel concepts progressively from installation to advanced features.

## Target Audience

University students learning web development with Laravel framework in an academic setting.

## Key Characteristics

- **Incremental Learning**: Each praktikum folder contains a complete Laravel project that builds upon the previous one
- **Indonesian Language**: All documentation, comments, and instructions are in Bahasa Indonesia
- **Self-Contained**: Each praktikum is an independent Laravel installation with its own dependencies
- **Academic Context**: Designed for classroom instruction with clear learning objectives and checklists

## Praktikum Sequence

1. Install Laravel - Basic setup and structure
2. Routing & Controller - Request handling and controllers
3. Blade Template - View templating with loops and data display
4. Master Template - Layout inheritance and template structure
5. Migration, Model, Seeder - Database fundamentals
6. Database Display - Eloquent ORM and queries
7. Database Relations - One-to-many, many-to-many
8. CRUD Create - Form input and validation
9. CRUD Edit & Delete - Update and delete operations
10. Validation - Form validation and error handling
