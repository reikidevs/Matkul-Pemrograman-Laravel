# Changelog - Praktikum 01

## Instalasi Awal Laravel

### File yang Dibuat
Semua file Laravel standar hasil dari:
```bash
composer create-project --prefer-dist laravel/laravel praktikum_laravel
```

### Struktur Folder Utama
- `app/` - Logika aplikasi
- `routes/` - Routing aplikasi
- `resources/views/` - View templates
- `database/` - Migration, seeder, factory
- `public/` - Assets publik
- `config/` - Konfigurasi aplikasi

### File Konfigurasi
- `.env` - Environment variables
- `.env.example` - Template environment
- `composer.json` - PHP dependencies
- `package.json` - JavaScript dependencies

### Fitur Bawaan Laravel
- ✅ Authentication scaffolding
- ✅ Blade template engine
- ✅ Eloquent ORM
- ✅ Artisan CLI
- ✅ Migration system
- ✅ Testing framework
