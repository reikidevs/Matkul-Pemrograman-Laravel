# Praktikum 01: Install Laravel

**Status:** ✅ Selesai

Tutorial instalasi Laravel dan pengenalan struktur folder dasar.

## 📖 Dokumentasi Lengkap

Baca dokumentasi lengkap di: **[docs/README.md](docs/README.md)**

## 🎯 Tujuan Pembelajaran

- Menginstall Laravel
- Memahami struktur folder Laravel
- Menjalankan Laravel development server
- Melihat halaman welcome Laravel

## 🚀 Quick Start

```bash
# Masuk ke folder praktikum
cd praktikum_01_install

# Install Laravel (jika belum)
composer create-project --prefer-dist laravel/laravel praktikum_laravel

# Masuk ke folder project
cd praktikum_laravel

# Jalankan server
php artisan serve
```

Buka browser: **http://localhost:8000**

## 📋 Persyaratan

- PHP 8.1 atau lebih tinggi
- Composer
- MySQL 8.0 atau lebih tinggi
