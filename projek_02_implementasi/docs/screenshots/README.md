# 📸 Folder Screenshot Laporan

Letakkan screenshot aplikasi di folder ini dengan nama file PERSIS seperti di bawah.
Script `generate_laporan.py` akan otomatis menyisipkan gambar ini ke laporan .docx.

Jika file belum ada, laporan tetap dibuat dengan kotak placeholder.
